About This Project

AgriPrice Intelligence is a multi-factor commodity price prediction system for Tamil Nadu mandis. It integrates real-time Agmarknet price data, OpenWeatherMap 3-day forecasts, festival demand patterns, and agricultural harvest cycles to deliver actionable BUY/SELL/HOLD advisories powered by Google Gemini AI.

Built for farmers, traders, and consumers across 38 Tamil Nadu districts covering 80+ commodities. Predictions use a 7-day sliding window moving average with graceful Gemini web-search fallback when live data is unavailable.
